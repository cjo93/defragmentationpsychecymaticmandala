# DEFRAG: The Kinetic Framework (Version 3.0 / Canonical Spec)

## CORE PHILOSOPHY
**Somatic Math / The Law of the Spin**
"No Parking on the Dance Floor."
Shift from "Healing" (Stillness/Garage) to "Integration" (Motion/Dance Floor).

## CANONICAL ENTRY SEQUENCE (NON-NEGOTIABLE)
1. **VOID**: True black, silence.
2. **ONION WHEEL**: Planetary wheel/rings. Rotates into vertical axis.
3. **SPRING**: Spiral journal representation.
4. **AXIS MUNDI**: Fly down the vertical axis.
5. **CYMATIC MANDALA**: Shifts and SNAPS into final form.
6. **USER GLYPH**: The mandala becomes the user's identity glyph.
7. **THE FLOOR**: Pull back to the checkerboard grid.

## VISUAL LAW
- **Background**: True Black (`#000000`).
- **Palette**: Monochrome base.
- **Indicators**: Cyan intensity (Calm/Coherent).
- **Exceptions**:
    - Iridescence allowed ONLY on user input, titles, and snap-to-form.
    - Gold allowed ONLY during "The Bow" closure.
- **Prohibited**: No Red.

## DATA & STATE (ENGINE)
- **Input**: Birth Data (DOB, Time, Location) + Current Time.
- **Output**: State Vector.
    - *Calm/Coherent*: Stable symmetry, low jitter, high cyan.
    - *Erratic*: High jitter, unstable form, low cyan.
    - *Parked*: Low contrast, near-zero motion.
    - *Tension*: Motion dynamics (no red).

## THE FLOOR (DASHBOARD)
- **Structure**: Checkerboard of tiles.
- **Content**: Each tile is a live-state mandala.
- **Camera**: Pan/Tilt/Zoom (Vortex/Axis/Spiral views).
